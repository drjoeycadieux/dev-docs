# 🚀 FAANG Dev Learning - Your Path to Top Tech Companies

[![Vue.js](https://img.shields.io/badge/Vue.js-3.5-4FC08D?logo=vue.js&logoColor=white)](https://vuejs.org/)
[![Vite](https://img.shields.io/badge/Vite-8.0-646CFF?logo=vite&logoColor=white)](https://vitejs.dev/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A comprehensive, modern learning platform designed to help aspiring software engineers master coding, algorithms, system design, and ace their technical interviews at **FAANG** (Facebook, Amazon, Apple, Netflix, Google) and other top tech companies.

## ✨ Features

### 🏠 Home Page
- **Hero Section** with compelling statistics showcasing 500+ coding problems, 50+ system design topics, and 100+ interview questions
- **Feature Cards** highlighting structured learning, hands-on practice, progress tracking, and expert guidance
- **Call-to-Action** buttons to start your journey immediately

### 🗺️ Learning Path
Follow a curated 5-stage learning journey designed by FAANG engineers:

| Stage | Title | Duration | Focus Areas |
|-------|-------|----------|-------------|
| 1 | **Foundation** | 2-3 months | Programming basics, DSA fundamentals, Big O |
| 2 | **Core Competency** | 3-4 months | Advanced DSA, algorithm patterns, OOP, SQL |
| 3 | **Advanced Topics** | 3-4 months | Complex DP, graph algorithms, system design |
| 4 | **Interview Prep** | 2-3 months | LeetCode practice, mock interviews, behavioral |
| 5 | **FAANG Ready** | Ongoing | Company-specific prep, negotiation, leadership |

### 📚 Topics
Interactive topic exploration with difficulty levels:
- **Data Structures**: Arrays, Linked Lists, Trees, Graphs, Hash Tables, Heaps
- **Algorithms**: Sorting, Searching, Two Pointers, Sliding Window, Backtracking
- **Dynamic Programming**: 1D/2D DP, Knapsack, LCS, MCM, Bitmask DP
- **System Design**: Load Balancing, Caching, Sharding, Microservices, APIs

Features include:
- Difficulty badges (Easy 🔵, Medium 🟡, Hard 🔴)
- Problem counts for each topic
- Progress tracking with visual indicators

### 🔗 Resources
Hand-picked resources across multiple categories:
- **Online Platforms**: LeetCode, HackerRank, Codeforces, GeeksforGeeks
- **Books**: Cracking the Coding Interview, CLRS, Clean Code, DDIA
- **Video Courses**: Coursera, CS50, Grokking System Design
- **Mock Interviews**: Pramp, Interviewing.io, Gainlo
- **Communities**: Reddit, Blind, Discord study groups

### 💼 Interview Preparation
Comprehensive guides for all interview types:

## 🚀 Getting Started

### Prerequisites
- Node.js v18 or higher
- npm v10 or higher

### Installation

```bash
# Clone or navigate to the project directory
cd faang-dev-learning

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run linter
npm run lint
```

The development server runs on `http://localhost:5173` by default.
